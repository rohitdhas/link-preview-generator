## 🌐 Link Preview API

Get link preview data in JSON format with this api

### Example GET Request:

```js
fetch(
  "https://link-preview-generator.vercel.app/api/getPreview?url=https://github.com"
)
  .then((res) => res.json())
  .then((data) => console.log(data));
```
